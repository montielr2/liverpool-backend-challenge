# liverpool-backend-challenge
El equipo de ventas del Puerto de Liverpool necesita un sistema que permita la gestión de pedidos realizados por la aplicación y por el sitio web.

## Tech stack
- Java 17
- Spring Boot
- MongoDB
- Maven

## Code Style

El proyecto fuerza la validación de estilo de código usando el plugin de Maven CheckStyle alineado a los lineamientos de Google Java Style.

Para validar:
- mvn checkstyle:check
